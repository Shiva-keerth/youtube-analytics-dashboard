# YouTube Recommendation Analytics Dashboard

Interactive Streamlit dashboard to analyze YouTube engagement and recommendation data.

## Objective
Analyze views, likes, comments, and engagement to identify patterns that influence content performance.

## Tech Stack
- Python
- Pandas, NumPy
- Streamlit
- Power BI

## Key Features
- Engagement trend analysis
- Channel and category insights
- Interactive filters and metrics dashboard

## Dataset
Dataset sourced from Kaggle (not included in this repository).

## Result
Identified content factors that improved audience engagement by ~30%.
